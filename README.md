# Liftline

A workout tracking app for logging exercises, sets, reps, and weights.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, React Router
- **Backend:** Express, PostgreSQL, JWT auth

## Getting Started

### Backend

```bash
cd backend
npm install
# Configure your .env with database URL and JWT secret
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
